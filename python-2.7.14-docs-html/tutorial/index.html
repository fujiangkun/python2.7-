Python 教程¶

Python是一门简单易学，功能强大的编程语言。 它拥有高级的数据结构以及简单有效的面向对象编程方法。Python优雅的语法、动态编程以及解释型语言的特性，使得它在大部分的平台上，在众多领域的脚本和快速应用开发上都是一门极佳的编程语言。

Python解释器以及标准库不论是源代码还是二进制形式都可以从Python官网的所有主要平台获取。 https://www.python.org/, 并且可以随意分享，官网还拥有发行版和丰富的免费第三方模块、程序和工具以及文档说明

Python解释器极易扩展，使用C或者C++（或者C语言可以调用的其他语言）实现新的功能或者数据类型。Python同样适合作为可定制程序的扩展语言。

此教程非正式地想读者介绍python语言和系统的基本概念和特性。它有助于进行python动手实践，所有的样例都包含此教程中，所以可以离线阅读。

如果你想获取Python标准对象和模块的介绍，请查阅Python标准库. Python语法手册将给出更详细的语法介绍。如果想用C语言或者C++编写扩展，请阅读 扩展与嵌入式Python解释器和 Python/C 接口参考手册. 这里还有几本书更深入地讲解python相关的知识。

此教程并不会全面去介绍Python的每一个特性，尤其是通用的特性，它只会介绍那些值得一提的特性，以便让你更好的了解Python的喜好与风格。 当你阅读完此教程，你将能自己阅读和编写模块和程序，之后你将会在 Python标准库学习更多不同的模块.
